# Fornax
*Axenide's alchemical furnace.*

**Fornax** is an opinionated Nix flake containing the tools I use for development. Currently:
- tmux
- fish
- starship
- NvChad
- lazygit
- zoxide
- yazi
Among some extras like `fastfetch`, `cava` and `ffmpeg`.

It also includes `bitwarden-cli` for secrets management.

This project is meant to work with `home-manager`, `nix profile add`, `nix profile run` and `nix develop`.
